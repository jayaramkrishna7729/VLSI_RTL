# VLSI_RTL
Implementation and simulation of various Digital circuits using XilinX ISE 14.7 and modelSim

List of RTL codes 

1 - HALF_ADDER 
2 - FULL_ADDER
3 - HALF_SUBTRACTOR
4 - FULL_SUBTRACTOR
5 - MUX 2:1, 4:1, 8:1 
6 - DEMUX 1:2, 4:1, 8:1 
7 - ENCODER
8 - DECODER
9 - FREQUENCY_DIVIDER
10 - RIPPLE_CARRY_ADDER
11 - 8-BIT ALU
12 - BCD TO 7-SEGMENT DISPLAY
